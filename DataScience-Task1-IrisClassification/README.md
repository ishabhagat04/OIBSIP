# Iris Flower Classification

## Task Overview
This project is part of my Oasis Infobyte internship. The goal is to classify Iris flowers into three species based on their measurements.

## Dataset
- Uses the classic Iris dataset
- Features: Sepal length, Sepal width, Petal length, Petal width
- Target: Setosa, Versicolor, Virginica

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib/Seaborn

## Approach
1. Data exploration and visualization
2. Train-test split
3. Model training (Logistic Regression, KNN, Decision Tree)
4. Model evaluation
